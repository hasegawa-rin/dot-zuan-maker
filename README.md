# DOT ZUAN MAKER
画像からドット絵の図案を生成し、マインクラフト（教育版）のMakeCode用JavaScriptコードやCSV形式で出力できるWebツールです。

## 主な機能
- 図案の生成: 画像をアップロードするだけで、即座に図案を生成します
- 色抽出: K-means法による減色とHSLソートによる見やすいパレット表示
- 手動修正: 生成された図案をドット単位で修正できます
- ズーム・座標表示: 拡大縮小や行列番号の表示ができます
- マインクラフト連携: エージェントが図案通りにブロックを置いてくれるJavaScriptコードを出力
- 背景トリミング: 上下の余計な余白を自動でカットして出力

## 使い方
1. 左パネルから画像を読み込みます
1. 横幅や色数、明るさを調整します
1. 必要に応じて、右側のパレットから色を選んで図案を直接クリックして修正します
1. 「マイクラ用コード」または「CSV」ボタンを押してデータを保存します。

## 技術仕様
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS)
- ブラウザ上ですべての処理を行うため、サーバーへの画像アップロードは発生しません
